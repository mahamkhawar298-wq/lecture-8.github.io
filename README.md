# Enchanted Garden | Theme Studio

An interactive theme studio where users transform the page ambiance with garden-inspired color schemes and font controls.

## Overview

A magical web application featuring a sidebar with 5 garden themes that dynamically change background color, text color, and font size of the central paragraph. Includes zoom controls, image toggle, and full reset functionality.

## Features

| Feature | Description |
|---------|-------------|
| 5 Garden Themes | Rose Garden, Lavender Mist, Sunflower Glow, Emerald Grove, Cherry Kiss |
| Dynamic Styling | Changes paragraph background, text color, and font size |
| Zoom Controls | Increase/decrease font size independently |
| Image Toggle | Show/hide the central flower image |
| Full Reset | Restores all original styles and image visibility |
| Active State | Visual highlight on selected theme |

## Tech Stack

HTML5 · CSS3 (Flexbox, Grid, Backdrop-filter) · JavaScript

## Quick Start

Open `index.html` in any modern browser.

## Interactions

- **Click any theme** → paragraph styling changes
- **Zoom In/Out** → adjust font size only
- **Hide/Show Image** → toggle flower image visibility
- **Reset All** → restore original styles + show image
